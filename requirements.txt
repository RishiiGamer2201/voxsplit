# Voice cloning (XTTS v2) runs in its OWN env — do NOT install coqui-tts here.
# It needs transformers < 5, which drags huggingface-hub to 0.36, but gradio
# 6.20 requires huggingface-hub >= 1.2: installing it here breaks the web demo.
# So VoxSplit shells out to an isolated env instead (same pattern as
# separate_mossformer2.py -> the clearvoice env). One-time setup:
#
#   conda create -y -n voxsplit-tts python=3.10
#   conda run -n voxsplit-tts pip install coqui-tts torch torchaudio torchcodec
#   conda run -n voxsplit-tts pip install "transformers<5"
#
# The demo auto-detects that env and enables the Voice Clone panel. Note: the
# XTTS v2 WEIGHTS are under the Coqui Public Model License (NON-COMMERCIAL),
# accepted via COQUI_TOS_AGREED in src/tts/voice_clone_cli.py.
